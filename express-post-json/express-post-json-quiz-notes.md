## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- What does the `express.json()` middleware do and when would you need it?
  express.json() is middleware that only parses json and only looks at requests where the content type is a json content type. you would need this when expecting json parsable code
